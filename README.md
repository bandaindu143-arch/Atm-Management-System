# ATM Management System

This project is a Java-based ATM Management System using Java Swing and MySQL.
It allows users to check balance, withdraw, deposit money, and change PIN securely.

## Features
- Secure login and authentication  
- Balance enquiry, deposit, and withdrawal  
- PIN change option  
- Simple interface  

## Technologies Used
- Java Swing  
- MySQL  
- JDBC  

## How to Run
1. Open in Eclipse  
2. Connect to MySQL  
3. Run the main file
